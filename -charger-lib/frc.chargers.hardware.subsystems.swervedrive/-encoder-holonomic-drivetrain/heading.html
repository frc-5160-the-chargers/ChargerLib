heading

open override val heading: Angle

The current heading (the direction the robot is facing). This is equivalent to the gyro heading if a gyro is specified; otherwise, the pose estimator will calculate the robot's heading using encoders and odometry.

If possible, specify a gyro, like a frc.chargers.hardware.sensors.imu.ChargerNavX, in order to get the best possible results.
