RobotPoseMonitor

interface RobotPoseMonitor : HeadingProvider

Represents a pose monitor which combines wheel odometry(and optionally vision data) to produce a robot pose.

This class should not be implemented by vision cameras or sensors, as their pose results are not always accurate, and thus the nullable parameters are necessary.
