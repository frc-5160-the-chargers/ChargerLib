waitFor

fun waitFor(timeInterval: Time): ParallelRaceGroup
Content copied to clipboard

Adds a command that does nothing for a specified timeInterval, then completes.

Useful if a delay is needed between two commands in a SequentialCommandGroup. Note that running this in parallel with other commands is unlikely to be useful.
