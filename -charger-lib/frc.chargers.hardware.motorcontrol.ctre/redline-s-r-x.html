redlineSRX

inline fun redlineSRX(deviceNumber: Int, encoderTicksPerRotation: Int = 1024, factoryDefault: Boolean = true, configure: TalonSRXConfiguration.() -> Unit = {}): ChargerTalonSRX

Represents a TalonSRX powering a redline/ CIM motor.

This function supports inline configuration using the "configure" lambda function, which has the context of a TalonSRXConfiguration.

You do not need to manually factory default this motor, as it is factory defaulted on startup, before configuration. This setting can be changed by setting factoryDefault = false.

// example
val motor = redlineSRX(canId = 6){ inverted = false }
