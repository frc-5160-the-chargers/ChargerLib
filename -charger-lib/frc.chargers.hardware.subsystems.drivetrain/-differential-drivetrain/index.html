DifferentialDrivetrain

interface DifferentialDrivetrain : Subsystem
Content copied to clipboard

An interface used to control differential drivetrains. A differential drivetrain is one based on two sides that move independently. This is similar to how a tank drives, although it must not necessarily be controlled using "tank controls."

The Kit of Parts drivetrain each team receives is a differential drivetrain.

See here for more on differential drivetrains, and particularly here for an explanation of the various drive modes.
