heading

open override val heading: Angle
Content copied to clipboard

The current heading (the direction the robot is facing).

This value is calculated using the encoders, not a gyroscope or accelerometer, so note that it may become inaccurate if the wheels slip. If available, consider using a frc.chargers.hardware.sensors.NavX or similar device to calculate heading instead.

This value by itself is not particularly meaningful as it may be fairly large, positive or negative, based on previous rotations of the motors, including from previous times the robot has been enabled.

Thus, it's more common to use this property to determine change in heading. If the initial value of this property is stored, the amount of rotation since that initial point can easily be determined by subtracting the initial heading from the current heading.
