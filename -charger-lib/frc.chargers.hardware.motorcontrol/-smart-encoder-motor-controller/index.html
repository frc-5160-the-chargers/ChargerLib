SmartEncoderMotorController

interface SmartEncoderMotorController : EncoderMotorController, TemperatureProvider, CurrentProvider, VoltageProvider

An interface that represents an EncoderMotorController that can measure it's recorded temperature, applied current, and applied voltage, as well as run closed loop control on the motor itself.
